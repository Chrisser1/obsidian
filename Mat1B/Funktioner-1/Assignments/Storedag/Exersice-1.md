#### Answer
For this answer see [[Functions]]. Only g is a function, as $f$ is trying to map 1 to both 0 and 2. The function $g$ has $im(g)=\{0,1,2\} \subset \{0,1,2,3\}$ and is therefore not surjective. As $g(0) = 1$ and $g(2) = 1$ the function is not injective.
